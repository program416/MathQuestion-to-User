# 티어 기준
0번 이상 레슨 : 1-F<br>
3번 이상 레슨 : 1-E<br>
6번 이상 레슨 : 1-D<br>
10번 이상 레슨 : 1-C<br>
15번 이상 레슨 : 1-B<br>
20번 이상 레슨 : 1-A<br>
<hr>
<b>25번 이상 레슨 시 2티어 진입가능.<br>2티어 중에서도 아이언,브론즈,실버,골드로 니뉜다</b><br>
<i>특히 2티어부터는 티어 점수가 생겨서 더 자세한 실력을 판단할 수 있다</i>
<hr>
<h2 style="color:rgb(180, 100,110)">티어 점수 0~199: 2-브론즈</h2>
이 티어는 이제 막 랭크를 시작한 유저가 있는 티어로 주로 (세자리수) + (두자리수), (10~29) * (1~9) 등이 나온다
<h2 style="color:rgb(100,100,100)">티어 점수 200~399: 2-실버</h2>
이 티어는 어느정도 사용법을 익힌 유저가 있는 티어로 주로 (네자리) + (두자리), (세자리)+(세자리), (두자리)*(한자리)가 나온다
<h2 style="color:rgb(230,230,0)">티어 점수 400~599: 2-골드</h2>
<b>초보의 마지노선.</b>
초보 단계는 이 단계에서 **끝난다**. 이 단계는 주로 (네자리) + (세자리)가 나오며 (두자리) * (10~19)도 출제될 수 있다.<br>
<span style="font-size:1.4rem;">이 단계에서 올라가면 3티어가 되며 3티어는 티어 점수 600이 되어야 올라갈 수 있다.</span>
