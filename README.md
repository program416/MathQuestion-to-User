# MathQuestion-to-User
유저의 티어에 맞는 수학문제 추천

# 티어 기준
0번 이상 레슨 : 1-F<br>
10번 이상 레슨 : 1-E<br>
20번 이상 레슨 : 1-D<br>
30번 이상 레슨 : 1-C<br>
40번 이상 레슨 : 1-B<br>
50번 이상 레슨 : 1-A<br>
<hr>
<b>60번 이상 레슨 시 2티어 진입가능.<br>2티어 중에서도 아이언,브론즈,실버,골드로 니뉜다</b><br>
<i>특히 2티어부터는 티어 점수가 생겨서 더 자세한 실력을 판단할 수 있다</i>

# 언어
백엔드 : Python<br>
프론트엔드 : html,cs,js
